# ALKE WALLET

Alke Wallet es una aplicación web simple que simula una billetera digital.  
Permite iniciar sesión, realizar depósitos, enviar dinero a contactos y visualizar el historial de transacciones.

---

## FUNCIONES PRINCIPALES

- **Login:** acceso con cualquier correo válido y contraseña de al menos 4 caracteres.
- **Menú principal:** muestra el saldo disponible y enlaces a las demás secciones.
- **Depósitos:** ingresar dinero y actualizar el saldo.
- **Enviar dinero:** seleccionar un contacto, enviar dinero y agregar nuevos contactos.
- **Autocompletar contactos:** búsqueda rápida de contactos guardados.
- **Historial de transacciones:** visualizar depósitos y transferencias recientes.
- **Mensajes dinámicos:** confirmaciones y alertas con jQuery y Bootstrap.


## TECNOLOGIAS UTILIZADAS

- **HTML5** y **CSS3**
- **Bootstrap 5** (estilos y componentes)
- **JavaScript (jQuery)** para la lógica
- **LocalStorage** para guardar datos en el navegador

## CÓMO USAR

- Abre index.html en el navegador.
- Ingresa cualquier correo válido y una contraseña de al menos 4 caracteres.
- Explora las funciones de depósito, envío y transacciones.
